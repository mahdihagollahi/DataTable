# DataTable

> a input recive the JSON file and show as table








## Installation

> First, clone the project

```bash
git clone  https://github.com/mahdihagollahi/DataTable.git
```

> to run DataTable with npm

```bash
cd my-project
  npm install
  npm run dev
```
